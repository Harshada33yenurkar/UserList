# User List App

A React Native app that fetches and displays users from a public API.

## Features
- Fetch & display users (name, email, phone, city)
- Search by name
- Navigate to user detail screen
- Loader & error handling

## Screens
- UserList– lists all users with search
- UserDetail – shows full user info

## Tech Stack
- React Native (JavaScript)
- React Navigation

## Setup
npm install
npx react-native run-android

## API
https://jsonplaceholder.typicode.com/users
